

# Telecom Customer Churn Prediction

## Project Overview
Predict which customers are likely to churn using machine learning. This helps telecom companies design retention strategies.

## Dataset
- Features: customer tenure, contract type, monthly charges, etc.
- Target: churn (0 = stay, 1 = churn)

## Models Used
- Logistic Regression (final model)
- Random Forest (comparison model)

## Steps
1. Data Cleaning & Preprocessing
2. Encoding Categorical Features
3. Handling Class Imbalance
4. Training Models (Logistic Regression & Random Forest)
5. Model Evaluation
6. Business Insights
